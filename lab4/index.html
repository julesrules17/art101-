Pseudocode for an everyday task

Task: serve a bow of salsa

  1. gather ingredients
    1. onions
    2. tomatoes
    3. cilantro
    4. jalepenos
    5. lime
    6. salt
    7. garlic
  2. prep ingredients
    1. wash and chop tomatoes
    2. dice onion
    3. dice jalepenos
    4. chop cilantro
    5. mince garlic
  3. mix ingredients
    1. combine tomatoes, onion, jalapenos, cilantro, lime juice, and garlic in a bowl
    2. add salt to taste
  4. taste and adjust if needed
  5. transfer salsa to a serving bowl and eat with tortilla chips

Pseudocode for a simple computer game

Task: create a checkers game

  1. Set up the game board
    1. create an 8x8 grid
    2. place black and white pieces on alternating squares
  2. Define player actions
    1. select a piece to move
    2. choose a valid destination for the piece
    3. move the piece to the destination
  3. Implement game logic
    1. validate player moves
    2. handle capturing opponent's pieces
    3. determine win or draw conditions
